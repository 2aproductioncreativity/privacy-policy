Suppression de compte et de données - TaskGuard

TaskGuard vous permet de supprimer votre compte et vos données directement depuis l'application, ou en nous contactant par e-mail.

Option 1 : Supprimer votre compte depuis l'application

1 Ouvrez l'application TaskGuard et connectez-vous.
2 Allez dans Paramètres (icône engrenage en bas de l'écran).
3 Faites défiler vers le bas et appuyez sur "Supprimer mon compte".
4 Confirmez la suppression en entrant votre mot de passe (ré-authentification requise pour des raisons de sécurité).
5 Votre compte et toutes vos données seront supprimés définitivement.

Option 2 : Supprimer vos données partiellement

Vous pouvez supprimer certaines données sans supprimer votre compte :

  • Supprimer un profil enfant : Paramètres → sélectionner l'enfant → Supprimer
  • Supprimer des tâches : depuis l'écran Tâches, supprimez les tâches individuellement
  • Supprimer des récompenses : depuis l'écran Récompenses, supprimez les récompenses individuellement
  • Supprimer le planning hebdomadaire : depuis le Planning, supprimez les tâches planifiées
  • Supprimer un appareil enregistré : depuis l'écran Appareils, supprimez un appareil

Option 3 : Demander la suppression par e-mail

Si vous ne pouvez pas accéder à l'application, envoyez un e-mail à l'adresse ci-dessous avec l'objet "Suppression de compte TaskGuard" et l'adresse e-mail associée à votre compte. Nous traiterons votre demande dans un délai de 30 jours.

Données supprimées

La suppression de compte est définitive et irréversible. Les données suivantes seront supprimées :
  • Compte utilisateur : adresse e-mail, pseudo, nom de famille, photo de profil
  • Données familiales : nom de famille, code d'invitation, planning hebdomadaire
  • Profils enfants : prénoms, âges, photos de profil, points
  • Tâches : toutes les tâches créées et l'historique des tâches complétées
  • Récompenses : catalogue de récompenses et historique des récompenses obtenues
  • Punitions : historique des punitions
  • Appareils : tous les appareils enregistrés
  • Données d'authentification : compte Firebase Authentication
Note : Si vous êtes le parent principal d'une famille, la suppression de votre compte entraînera la suppression de toutes les données de la famille, y compris les données des parents invités associés. Les données d'abonnement gérées par Google Play ne sont pas affectées par la suppression du compte TaskGuard. Pour annuler votre abonnement, rendez-vous dans les paramètres de votre compte Google Play.

Durée de conservation après suppression

Toutes les données sont supprimées définitivement de nos serveurs (Firebase) dans un délai maximum de 30 jours après la demande de suppression.
